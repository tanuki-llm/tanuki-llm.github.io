2025 Project Report
 Code  Paper  Blog  Summary  Demo

TL;DR

Tanuki は、公募型の貢献者コミュニティによって開発された日本語特化の LLM です。Retrieval‑Augmented Generation (RAG) や 合成データ生成パイプラインを活用し、対話・作文の品質を GPT‑3.5 と同等レベルまで向上させました。 オープンライセンス (Apache 2.0) でモデル・学習コードを公開し、学術研究および産業応用を加速します。

👍 改善点

RAG により最新ニュースも正確に引用しながら回答可能。

和歌・俳句生成において BLEU が 18% → 32% に向上。

👎 制限事項

140 文字以上の長文要約では一部事実混入が生じることがある。

多言語 (特に中文) での推論精度は GPT‑3.5 に未到達。
